Image Search Abstraction

User Stories

  • User Story: I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • User Story: I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • User Story: I can get a list of the most recently submitted search strings.

Instructions

  1. In order to search for images, go to the path .../api/imagesearch/'query' where you replace 'query' with your search query
  2. If you want to paginate, add a ?offset=XXX where you replace XXX with where you want your results to start
  3. If you want to review recent queries, go to the path .../api/latest/imagesearch
